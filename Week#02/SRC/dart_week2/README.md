Pada Week 2 dipelajari dasar-dasar pemrograman menggunakan Dart yang mencakup pembuatan dan penggunaan variabel, tipe data seperti `int`, `double`, `String`, dan `bool`, serta berbagai operator seperti aritmatika, relasional, logika, assignment, ternary, dan null-aware, termasuk penggunaan `final`, `const`, dan konsep null safety agar kode lebih aman dan terstruktur. Selain itu, dikerjakan beberapa studi kasus seperti konversi suhu, kalkulator sederhana, validasi data pengguna, perhitungan diskon, dan penentuan status kelulusan menggunakan Map, serta tugas praktikum berupa program BMI, konversi mata uang, penerapan semua jenis operator dalam satu aplikasi, dan tantangan tambahan membuat aplikasi konversi unit dengan menu serta validasi input.
